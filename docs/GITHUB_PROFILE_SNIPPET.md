## Featured project

### [Qt Npcap UDP Fallback](https://github.com/ArashShafizadeh/qt-npcap-udp-fallback)

A Qt 6 and C++ hybrid UDP receiver for Windows that combines `QUdpSocket` with an Npcap raw Ethernet fallback. It includes IPv4/UDP parsing, VLAN and QinQ support, BPF filtering, adapter discovery, duplicate suppression, a Qt Widgets dashboard, and unit tests.
