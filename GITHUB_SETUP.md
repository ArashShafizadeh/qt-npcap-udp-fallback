# GitHub setup

Repository name:

```text
qt-npcap-udp-fallback
```

Description:

```text
A Qt 6 hybrid UDP receiver that uses Npcap as a raw-frame fallback when Windows drops packets before QUdpSocket.
```

Recommended visibility: **Public**, after confirming you have the right to publish the generic implementation.

Recommended topics:

```text
qt qt6 cpp udp npcap pcap wireshark networking packet-capture windows
```

Suggested first commit message:

```text
feat: add Qt and Npcap hybrid UDP receiver demo
```

After adding this folder in GitHub Desktop:

1. Review the **Changes** tab.
2. Confirm that no build output or private file appears.
3. Enter the suggested commit message.
4. Click **Commit to main**.
5. Click **Publish repository**.
6. Choose **Public** and publish.
